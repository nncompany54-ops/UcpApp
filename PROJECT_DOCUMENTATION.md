# توثيق مشروع تطبيق "المؤسسة المتحدة" (UCP App) 🚀

تم إعداد هذا المستند ليكون مرجعاً شاملاً لكل ما تم إنجازه في التطبيق منذ بداية المشروع وحتى الآن. يحتوي هذا الملف على الخطة الكاملة، هيكلية المشروع، والتقنيات المستخدمة.

---

## 1. نظرة عامة على المشروع
تطبيق "المؤسسة المتحدة لللأدوية والمستلزمات الطبية - قسم التجميل" هو منصة إلكترونية (تطبيق ويب + تطبيق أندرويد) يهدف إلى عرض المنتجات الطبية والتجميلية للعملاء، بالإضافة إلى عرض أحدث العروض (Banners)، مع توفير وسيلة تواصل سريعة ومباشرة عبر WhatsApp لطلب المنتجات والاستفسار عنها. كما يتضمن التطبيق نظام إشعارات ذكي لتنبيه المستخدمين بالعروض الجديدة.

---

## 2. الهيكلية التقنية (Architecture)
تم بناء المشروع باستخدام بنية منفصلة (Backend و Frontend) لضمان الأداء العالي والقابلية للتطوير:

### أ. الواجهة الخلفية (Backend) - لوحة التحكم ومزود البيانات
- **التقنية:** Python / Django & Django REST Framework.
- **قواعد البيانات:** SQLite (قابلة للترقية إلى PostgreSQL مستقبلاً).
- **المهام الرئيسية:**
  - إدارة المنتجات (Products) والأقسام (Categories).
  - إدارة العروض واللافتات الإعلانية (Banners).
  - استقبال وتخزين رموز الأجهزة (FCM Tokens) لإرسال الإشعارات.
  - إرسال الإشعارات (Push Notifications) آلياً إلى جميع الأجهزة عند إضافة عرض جديد (عبر مكتبة `firebase_admin`).

### ب. الواجهة الأمامية (Frontend) - تطبيق المستخدم
- **التقنية:** Flutter (Dart).
- **المنصات المدعومة:** 
  - تطبيق ويب (Web App) متوافق مع متصفحات الهاتف والكمبيوتر.
  - تطبيق أندرويد (Android App) - APK & AAB.
- **الشاشات الرئيسية:**
  - **شاشة البداية (Splash Screen):** شاشة ترحيبية تظهر عند فتح التطبيق بشعار المؤسسة وخلفية زرقاء داكنة جذابة.
  - **شاشة تسجيل الدخول (Login Screen):** واجهة مبدئية مصممة بأناقة، (تم تصميمها حالياً كواجهة تخطي للمستخدم، ويمكن تفعيل الربط برقم الهاتف مستقبلاً).
  - **الشاشة الرئيسية (Home Screen):** 
    - سلايدر لعرض أحدث العروض الإعلانية (Banners).
    - قائمة تفاعلية بالأقسام والمنتجات.
    - زر عائم للربط المباشر بـ WhatsApp للحجز.
    - ميزة ظهور تنبيه (Dialog) عند تشغيل التطبيق لعرض الإعلانات الجديدة التي لم يرها المستخدم من قبل، مع إمكانية التفاعل معها أو طلبها عبر واتساب.
  - **شاشة الإعدادات:** للتحكم في تفعيل أو إيقاف استلام الإشعارات.

---

## 3. البنية التحتية والاستضافة (Infrastructure & Deployment)
تم رفع ونشر التطبيق على خوادم فعلية ليصبح متاحاً للمستخدمين:

1. **استضافة Backend وتطبيق الويب (Hostinger VPS):**
   - تم إعداد الخادم (VPS) باستخدام نظام Ubuntu.
   - تم ضبط `Nginx` كخادم ويب و `Gunicorn` لتشغيل Django.
   - تم ربط وتفعيل شهادة الأمان (SSL / HTTPS) لضمان أمان البيانات.
   - تم نشر تطبيق الويب (Flutter Web) كنسخة إنتاج على نفس الخادم ليتمكن العملاء من التصفح بدون تحميل التطبيق.

2. **خدمة الإشعارات (Firebase Cloud Messaging - FCM):**
   - تم إنشاء مشروع `ucp-platform` على منصة Google Firebase.
   - تم دمج إعدادات Firebase بسلاسة في الكود البرمجي لتتوافق مع كل من الويب (Web) والأندرويد (Android) من خلال ملف `firebase_options.dart` الموحد، لتجاوز أخطاء التهيئة.

3. **متجر جوجل بلاي (Google Play):**
   - تم إنشاء حساب مطور شخصي لدى Google.
   - تم إنشاء مفتاح تشفير رقمي (Digital Keystore) بكلمة مرور `ucp123456` لضمان أمان وملكية التطبيق.
   - تم استخراج نسخة المتجر الرسمية `app-release.aab`.

## 4. تصميم الهوية والأيقونة
- تم سحب شعار "المؤسسة المتحدة" الأصلي، ومعالجته برمجياً لتفريغ الخلفية البيضاء وجعلها شفافة لتبدو احترافية.
- تمت إضافة نصوص واضحة بخط عربي متناسق (Tajawal) تحت الأيقونة.
- تم ضبط الأيقونة لتتوافق مع معايير تطبيقات الهاتف (Android Launcher Icons).

---

## 5. تسلسل الإنجازات (Timeline)
1. 🟢 **المرحلة الأولى:** بناء الـ Backend، إنشاء النماذج وقواعد البيانات، وبرمجة واجهات الـ API.
2. 🟢 **المرحلة الثانية:** بناء الـ Frontend وتصميم الشاشات، وربط التطبيق ببيانات السيرفر عبر الـ API.
3. 🟢 **المرحلة الثالثة:** إضافة ميزات تفاعلية متقدمة مثل (النقر للحجز عبر واتساب، ونوافذ الإعلانات المنبثقة التلقائية - Popup Banners).
4. 🟢 **المرحلة الرابعة:** إعداد السيرفر (Hostinger) ورفع الواجهة الخلفية وتطبيق الويب ليعملا بشكل حي.
5. 🟢 **المرحلة الخامسة:** دمج Firebase، تفعيل الإشعارات للموبايل والويب، وحل تعارضات المنصات المتعددة.
6. 🟢 **المرحلة السادسة:** تصميم أيقونة التطبيق، تجهيز الحزم الأمنية (Keystore)، واستخراج النسخ النهائية للتشغيل المباشر (APK) وللرفع على المتجر (AAB).
